# 🚀 Tanvixa

> **Discover Smart Tech. Buy with Confidence.**

Tanvixa is a modern tech gadget discovery platform built for a global audience. Users can search products using product codes from our videos, explore the latest gadgets, read buying guides, and access trusted buying links from supported affiliate partners.

# 📦 Product System

Every product is managed from the JSON database.

Each product contains:

- Product Code
- Product Name
- Product Image
- Description
- Features
- Affiliate Link

The website automatically loads products from the database.
